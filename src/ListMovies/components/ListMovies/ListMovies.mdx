# ListMovies

`ListMovies` é um componente que pode ser utilizado para listar os filmes que foram classificados como melhores do ano.<br/>
A lista será filtrada pelo ano que será passado como parâmetro na barra de busca e também pelo seletor de "Winner".<br/>

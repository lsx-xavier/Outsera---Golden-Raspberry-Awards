# Table

`Table` é um componente que pode ser utilizado para criar um container que tem uma sombra e bordar, para que seja possível adicionar um estilo de layout.

    ##### Explicação dos types

    ```ts
      type ColumnsProps = {
        id: string;
        value: string | React.ReactNode;
      }[];

      type RowsProps = {
        id: string;
        columnsValues: ColumnsProps;
      }[];

      type TableProps = {
        columns: ColumnsProps;
        rows: RowsProps;
        isLoading?: boolean;
        rowsLoading?: number;
      };
    ```
